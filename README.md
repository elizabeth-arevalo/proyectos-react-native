# Documentación del Repositorio: **proyectos-react-native**

Este repositorio contiene varios proyectos desarrollados en React Native, cada uno enfocado en diferentes aspectos del desarrollo móvil utilizando esta tecnología.

---

## Proyectos Incluidos

### 1. **01-react-foundation/**

#### Descripción

Este proyecto contiene las bases de desarrollo en React y TypeScript, proporcionando un punto de partida sólido para cualquier aplicación de React Native.

---

### 2. **02-Calculator App/**

#### Descripción

Una aplicación de calculadora interactiva desarrollada en React Native. La aplicación permite a los usuarios realizar las siguientes operaciones matemáticas:

-   Suma
-   Resta
-   Multiplicación
-   División
-   Cambio de signo
-   Eliminación del último dígito

#### Características

-   **Interfaz interactiva:** Fácil de usar, con botones osuros y funcionalidades intuitivas.
-   **Operaciones matemáticas básicas:** Realiza cálculos matemáticos comunes.
-   **Manipulación de números:** Permite cambiar el signo de un número y eliminar el último dígito.

---

### 3. **Navigation App/**

#### Descripción

Una aplicación que explora diferentes tipos de navegación disponibles en React Native, utilizando stack navigation, drawer navigation, y handlers de gestos.

#### Características

-   **Múltiples tipos de navegación:** Uso de navegación por pila (stack) y cajón (drawer).
-   **Interactividad mejorada:** Uso de handlers de gestos y animaciones personalizadas.
-   **Iconos:** Integración de iconos personalizados para una mejor experiencia de usuario.
